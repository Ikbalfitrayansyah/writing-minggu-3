JavaScript - Array


Array yaitu mengorganisaikan atau mengelompokkan data

Array adlah tipe data yang dapat menyimpa tipe data lainnya seperti string, number dan lainnya


mengakses array

  • Array didefinisikan menggunakan square brackets
  • Array pada javascript dihitung dari index data ke-0

const in Array

  • Jika menggunakan let, kita dapat mengubah array dengan array baru dan konten nilai yang ada di dalam array dengan nilai lain
  • Const tidak bisa melakukan update data. Namun pada Array kita dapat melakukan update konten nilai di dalam array (mutable).
  • Yang tidak bisa adalah mengubah array dengan array yang baru jika menggunakan const

Array properti

Array memiliki 5 properti yang sering digunakan yaitu

  • constructor
  • length
  • index
  • input
  • prototype

Array method

  • push(), untuk menambahkan item array pada urutan yang paling akhir.
  • pop(), untuk menghapus item array index terakhir
  • unshift(), menambahkan item Array pada index pertama
  • sort(), mengurutkan secara Ascending atau Descending Alphanumeric

Looping pada array

  • looping.map => melakukan perulangan/looping dengan membuat array baru
  • loping.each => untuk melakukan looping pada setiap elemen array.

JavaScript Regex

adalah susunan karakter/deretan karakter spesial yang menggambarkan pola untuk pencarian text pada sebuah string atau document

  • Literals => konsep regex yang paling sederhana dimana kita membuat regex sesuai dengan text yang ingin kita cari
  • test => mengembalikan nilai BOOLEAN (TRUE/FALSE) untuk kecocokan sebuat text yang dicari.
  • match =>mengembalikan nilai array dari karakter yang match.
  • flag => i = Untuk menghandle case-sensitive. Tidak mempermasalahkan besar kecilnya sebuah karakter. Tidak membedakan antara A dan a. g = Untuk mencari kedalam seluruh string yang ingin dicari. Jika tidak menggunakan flags g, maka sistem akan mengembalikan nilai array pertama yang ditemukan saja atau tidak melanjutkan pencarian.

Karakter set short syntax

  • \d : Seluruh number/digit character. Contohnya [0-9]
  • \w : Alphanumeric character (26 huruf abjad dan angka). Contohnya [A-Za-z0-9_]
  • \s : Whitespace character (space, tab, newline, and similar). Contohnya [\t\r\n\f\v]

Quantifiers

  • Quantifiers adalah membuat format diatas menjadi lebih baik dan rapih
  • Quantifiers ditandai dengan format curly braces {}
  • \w{3} : akan match dengan 3 kata/word karakter
  • \w{4,7}: akan match minimum 4 kata/word character dan maksimal 7 kata/word character

JavaScript - OOP (object oriented programing)

OOP adalah principle yang diterapkan pada bahasa programming Javascript

4 pilar pada OOP

  • Encapsulation
  • Abstraction
  • Inheritance
  • Polymorpishm

contoh oop pada javascript


this keyword

  • This adalah sebuah keyword khusus yang merujuk pada pada objek pemiliknya
  • nilai dari this sangat bergantung pada di mana keyword this ini diletakkan atau di panggil
  • Jika this di dalam sebuah function maka ia merujuk pada global objek atau window

Encapsulation

cara untuk membatasi akses langsung ke properti atau method dari sebuah objek


Inheritance

sebuah proses dimana sebuah class mewariskan property dan methodnya ke class lain atau childnya


Polymorpishm

method yang diwariskan bisa kita ubah dengan behaviour yang berbeda menyesuaikan child class yang kita buat.


Abstraction

menyembunyikan detail tertentu dari sebuah objek/method dan hanya menampilkan fungsionalitas atau fitur penting dari objek tersebut
